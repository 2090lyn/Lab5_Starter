1) Would you use a unit test to test the “message” feature of a messaging application? Why or why not? For this question, assume the “message” feature allows a user to write and send a message to another user.
   Yes, because parts of the message feature (like creating or validating a message) can be tested in isolation with a unit test. However, the full end-to-end process of sending a message to another user would not be fully covered by a unit test since it involves multiple interacting systems.
